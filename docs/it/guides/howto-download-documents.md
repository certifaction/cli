# Scaricare Documenti

Per scaricare documenti, utilizzare l'endpoint [`/download`](../references/api#get-/download).

_Esempio:_

```shell
curl --location --request GET 'http://localhost:8082/download?file=REQUEST_URL' \
--header 'Authorization: PERSONAL_API_KEY'
```
