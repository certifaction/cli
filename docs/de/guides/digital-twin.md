# Certifaction Digital Twin

Sie können Ihre gedruckten Dokumente mit einem [Digital Twin](https://certifaction.com/digital-twin/) sichern.
Digital Twins verbinden gedruckte Dokumente mit ihren digitalen Originalen.
Auf diese Weise können Sie die sicher gespeicherte digitale Kopie auf Ihr Gerät herunterladen, ihre Authentizität überprüfen
und alle Dokumentbetrugsrisiken eliminieren.

So funktioniert der Digital Twin:

1.  Vor der Signatur wird eine eindeutige und sichere URL erstellt und dem Dokument mit einem QR-Code hinzugefügt.
2.  Das Dokument wird verschlüsselt.
3.  Die verschlüsselte Kopie wird im Certifaction Digital Archive (DA) gespeichert.
4.  Die Digital Twin-URL wird im Certifaction-Dokumentlocator aufgezeichnet.
